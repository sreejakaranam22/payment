# Project IOM API

Instruction to run locally

```termnal
npm install
npm start
```

Madatory envoronment variables

```JavaScript
PORT=
MONGO_URI=
NODE_ENV=
```

The .env file for development need to have these values

```JavaScript
NODE_ENV=development

// express app
PORT=3000
HOST=
HOST_URL=

// database
MONGO_URI =

// email
SMTP_PASSWORD =
SMTP_FROM_NAME =
SMTP_EMAIL =
SMTP_HOST =
SMTP_FROM_EMAIL =
```

The default NODE_ENV is 'development'
